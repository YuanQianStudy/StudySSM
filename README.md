# StudySSM

这是萌新在学习SSM框架时跟随bilibili的up主@遇见狂神说 做的一个极简图书管理系统。
在其基础上完成了老师布置的课后作业。
使用拦截器实现用户登录判断。
使用Ajax验证用户名密码是否正确。
使用Restful风格的URL地址

**开发工具：**

idea 2020.3

**开发框架：**

Spring 5.3.9

SpringMVC 5.3.9

Mybatis 3.5.7

**其它依赖：**

mysql 8.0.26

druid 1.2.8
